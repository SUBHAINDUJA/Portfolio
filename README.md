## ğŸš€ Usage

### Customization

1. **Personal Information**: Update details in `src/components/Home/Home.js`
2. **Projects**: Modify project data in `src/components/Projects/Projects.js`
3. **Skills**: Update tech stack in `src/components/About/Techstack.js`
4. **Resume**: Replace resume file in `src/Assets/`
5. **Styling**: Customize colors and styles in `src/style.css` and component-specific CSS files

### Building for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.
